# The Aletheia Protocol

> *A new method of learning — no reading, no lectures, no curriculum. Just your mind, an AI, and what gets built between them.*

---

Most learning tools optimize around the text — they summarize it, gamify it, quiz you on it. The Aletheia Protocol discards the text entirely. The AI already holds the knowledge. The conversation is the curriculum. Understanding is not delivered. It is uncovered.

The method runs in two acts: first, your existing beliefs about a topic are surfaced and stress-tested until the cracks become visible. Then, from your own foundations, a more accurate model is built — through dialogue, not explanation. You never read a summary. You never receive a lecture. You arrive at the idea yourself, or you don't arrive at all.

This repository contains everything needed to understand and run the protocol.

## How to run a session

1. Open `prompt.md`
2. Copy the contents of the **SYSTEM PROMPT** block
3. Paste it into the system prompt field of Claude (recommended) or any instruction-following AI
4. Start the conversation
5. The session begins with four words

That's it. No setup. No reading. No preparation required from the learner.
